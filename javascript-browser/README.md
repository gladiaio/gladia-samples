# JavaScript - Browser

Browser samples for [Pre-recorded](https://docs.gladia.io/api-reference/pre-recorded-flow) and [Live](https://docs.gladia.io/api-reference/live-flow) transcription. No build step: open the HTML files in a recent browser and submit the form.

**API key:** Get your Gladia key from [app.gladia.io](https://app.gladia.io) and enter it in the form when running the samples.

All samples are also deployed at: **https://gladiaio.github.io/gladia-samples/**

## Core concepts

### Pre-recorded

Documentation: [Pre-recorded flow](https://docs.gladia.io/api-reference/pre-recorded-flow)

Transcribe an audio or video file (URL or upload).

- **Run locally:** Open `src/pre-recorded.html` in a browser and submit the form.
- **Online:** [pre-recorded.html](https://gladiaio.github.io/gladia-samples/pre-recorded.html)

### Live

Documentation: [Live flow](https://docs.gladia.io/api-reference/live-flow)

**Live from microphone** — real-time transcription from your microphone.

- **Run locally:** Open `src/live-from-microphone.html` in a browser and submit the form.
- **Online:** [live-from-microphone.html](https://gladiaio.github.io/gladia-samples/live-from-microphone.html)

**Live from file** — simulate a live session using an audio file.

- **Run locally:** Open `src/live-from-file.html` in a browser and submit the form.
- **Online:** [live-from-file.html](https://gladiaio.github.io/gladia-samples/live-from-file.html)
